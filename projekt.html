Run N' Fun

Ismertető

A Run ’N Fun projektet a régi flash játékok ihlették. Egy alapvetően egyszerű 3 dimenziós ügyességi reflex játék melyben a design legalább annyi szerepet kap mint a játék mechanikák. A játék egy nagy városban fog helyet kapni ahol egy betörőt irányíthatunk aki épp menekül a hatósági rendektől. A játék változatossága miatt a játék előre haladásával kijutunk a városból és egyre változatosabb környezetben folytathatjuk főhősünk menekülését.
